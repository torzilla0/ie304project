# ie304-project

METU-IE Summer Practice RAG Chatbot

## Overview
Intelligent RAG chatbot for METU Industrial Engineering Summer Practice queries.

- TF-IDF based retrieval (zero API cost)
- 27 FAQ entries + 30+ company database
- Turkish/English multi-lingual support
- Next.js + React + Google Generative AI

## Setup
```bash
npm install
echo 'GOOGLE_API_KEY=your_key' > .env.local
npm run dev
```
